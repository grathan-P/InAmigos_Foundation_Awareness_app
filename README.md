# InAmigos Foundation Awareness App

This is the InAmigos Foundation Awareness App, built with React and Vite to help people quickly discover the foundation's mission, explore its community work, and get involved through volunteering and outreach.

## Live Demo

- [amigos_foundation](https://in-amigos-foundation-awareness-app.vercel.app/)

## Overview

- Highlights the foundation's impact, programs, and community-driven initiatives in one place.
- Makes it easy for visitors to learn about ongoing projects, explore photo stories, and connect with the team.
- Encourages action by guiding users toward volunteering, sharing, and reaching out for support or collaboration.
- Built as a modern single-page React application.
- Deployed and accessible online through Vercel.

## Key Impact

- Raises awareness about the foundation's mission and the communities it serves.
- Creates a clean digital presence that builds trust and visibility.
- Makes engagement simple for supporters, volunteers, and curious visitors.
- Brings together information, visuals, and calls to action in one streamlined experience.
